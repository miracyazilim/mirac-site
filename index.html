HTML Nedir?

HTML (HyperText Markup Language), web sayfalarının iskeletini oluşturan işaretleme dilidir. Tarayıcının neyi nasıl göstermesi gerektiğini belirler.

HTML ile Neler Yapılır?

  • Başlık, paragraf, resim gibi içerikler eklenir
  • Bağlantılar (linkler) oluşturulur
  • Sayfa yapısı belirlenir

Basit Bir HTML Örneği


<!DOCTYPE html>
<html>
  <head>
    <title>Benim İlk Web Sayfam</title>
  </head>
  <body>
    <h1>Merhaba Dünya!</h1>
    <p>Bu, ilk HTML sayfamdır.</p>
  </body>
</html>
